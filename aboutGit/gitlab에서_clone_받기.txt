1. lab.hanium.or.kr 접속

2. 로그인

3. 슬랙에 한결형이 올려주신 url 복사하여 브라우저에 접속

4. subgroups and projects 목록의 haniumSpeechProject 클릭

5. 화면 navbar 아래 왼쪽상단 clone버튼 클릭하여 http주소 복사

6. clone하여 폴더를 만들 경로에서 ctrl+우클릭 - git bash here 선택

7. git clone 입력 후 Shift + Insert 하여 아까 복사한 gitlab프로젝트 주소 붙여넣기

8. enter 치고 clone 시작. 폴더를 제대로 받아왔는지 확인, clone완료

9. 에러는 구글링으로 해결할 수 있읍니다 화이팅
